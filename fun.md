# Hugo Book 知识库项目功能概览

## 1. 内容管理与同步 (Workflow)
- **Obsidian 集成**：通过 `deploy.sh` 脚本，实现从 Obsidian 知识库到 Hugo 项目的自动化同步。
- **单向同步逻辑**：以 Obsidian 为源 (Source of Truth)，自动处理文章新增、修改及物理删除。
- **附件同步**：支持自动同步 Obsidian 中的图片、PDF 等附件到 Hugo 的静态资源目录。

## 2. 核心技术特性 (Technical Features)
- **主题框架**：基于 `hugo-book` 主题，提供类 GitBook/Wiki 的文档导航体验。
- **双模内容架构**：
    - **Docs 模式**：用于结构化的技术文档和知识树（位于 `content/docs`）。
    - **Blog 模式**：用于时间线式的随笔与记录（位于 `content/posts`）。
- **数学公式**：集成 `KaTeX`，支持高效的 LaTeX 数学公式渲染。
- **图表支持**：开启 `Mermaid` 渲染，支持通过代码生成流程图、时序图等。
- **全站搜索**：内置 `BookSearch`，实现基于本地索引的即时搜索功能。

## 3. 自动化部署 (CI/CD)
- **Vercel 云端构建**：与 GitHub 仓库联动，实现代码提交即自动触发全球 CDN 部署。
- **环境隔离**：通过 `vercel.json` 指定 Hugo 编译版本及子模块递归初始化，确保生产环境一致性。

## 4. UI/UX 体验
- **响应式设计**：完美适配桌面、平板及移动端。
- **自动深色模式**：支持根据系统偏好自动切换明亮/深色主题。
- **多语言支持**：底层具备 i18n 多语言扩展能力。

## 5. 自定义扩展
- **布局定制**：针对 `posts` 列表页进行了自定义布局优化（`layouts/posts/list.html`）。
- **样式覆盖**：通过 `assets/_custom.scss` 方便地进行 CSS 样式微调。

## 6. Post 内容具体样式 (UI Style Guide)

项目的文章页面采用了高度定制化的 **环境式玻璃拟态 (Ambient Glassmorphism) 2.0** 设计风格，主要特征如下：

### 视觉基础
- **背景设计**：全站使用多层饱和度较高的渐变背景（淡紫、淡蓝、淡粉、淡灰紫），并叠加了径向渐变纹理，营造出一种漂浮感。
- **磨砂效果**：通过 `backdrop-filter: blur(30px) saturate(180%)` 实现极致的磨砂玻璃感。

### 博客列表 (Card Layout)
- **玻璃卡片**：每篇博文在列表页显示为独立的半透明卡片，具有 `24px` 的大圆角和多层细腻阴影。
- **悬停动画**：卡片在鼠标悬停时会发生位移 (`translateY(-4px)`) 且阴影加深，增强交互反馈。
- **全卡片点击**：通过 `.book-post-overlay-link` 实现整个卡片区域可点击。

### 正文与排版 (Typography)
- **专属字体**：
    - **标题**：采用 `Comfortaa` 圆润字体，去除下划线装饰，整体风格更加现代和亲和。
    - **正文**：采用 `IBM Plex Serif` 衬线体，优化阅读长文本的舒适度。
- **引言 (Blockquote)**：半透明蓝色背景，配合 `8px` 模糊，侧边辅以 4px 的蓝色装饰条。
- **表格 (Table)**：半透明白色背景，表头采用 `100, 150, 255` 浅蓝底色，行间支持悬停高亮。

### 代码块美化 (Code Blocks)
- **IDE 风格**：代码块不再使用深色背景，而是采用类似 VSCode Light 的浅色系风格。
- **语法高亮**：
    - 关键词/关键字：`#d73a49` (红色)
    - 字符串：`#032f62` (深蓝)
    - 注释：`#6a737d` (灰色斜体)
    - 函数名：`#6f42c1` (紫色)
- **行内代码**：淡蓝色背景 (`rgba(100, 150, 255, 0.18)`) 配合深蓝色文字，带有精致的边框。
- **列表页图片隐藏**：为了保持列表页整洁，强制隐藏了摘要 (`.book-post-summary`) 中的所有图片，只保留纯文本预览。

### 响应式细节
- **移动端优化**：在手机端自动降低背景纹理强度，并将主内区域模糊度调整为 `20px`，确保性能与美观的平衡。
